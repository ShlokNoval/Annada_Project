# 🌱 Annada – AI-Powered Farmer Assistance App

Annada is a Flutter-based mobile application designed to empower marginal and small-scale farmers using modern technology. The app integrates AI, real-time data, and government resources to help farmers make better agricultural decisions and improve productivity.

---

## 🌱 Features

### 🔐 User Authentication
- Google Sign-In
- Email / User ID & Password login
- Persistent login (auto-login on app restart)

### 🌦️ Live Weather Updates
- Real-time temperature and weather conditions
- Location-based weather detection
- Weather alerts for sudden changes

### 🌿 Crop Image Detection (AI)
- Scan crops to analyze plant health
- AI-based suggestions and remedies

### 🧮 Fertilizer Calculator
- Supports Acre & Hectare units
- Crop-specific fertilizer recommendations
- Modern and easy-to-use UI

### 📚 Farming Tutorials
- Text, image, and video-based tutorials
- Practical guidance for farmers

### 🏛️ Government Yojna Information
- Indian government schemes for farmers
- Scheme details with official registration links

### 🤖 AI Chatbot (Gemini Integration)
- Smart chatbot for farming-related queries
- Instant AI-based assistance
